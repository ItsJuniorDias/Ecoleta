### Installing

**Cloning the Repository**

```
$ git clone https://github.com/ItsJuniorDias/Ecoleta.git

$ cd Ecoleta
```

**Installing dependencies**

```
$ yarn install
```

_or_

```
$ npm install
```
**Running**

**Backend**

```
$ cd backend
$ yarn dev
```

**Frontend**

```
$ cd frontend
$ yarn start
```
